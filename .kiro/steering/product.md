# Product Overview

社内向け備品管理システム（Equipment Management System）。
備品の登録・貸出・返却・在庫管理を一元化し、担当者の運用負荷を削減する。

## Core Capabilities

- **備品管理**: 備品の登録・編集・ソフトデリート。カテゴリ・ステータス（available / in_use / repair / disposed）・在庫数を管理。
- **貸出・返却フロー**: ユーザーが申請 → 管理者が承認 → 返却処理。排他ロックで同時申請の競合を防止。
- **在庫・延滞アラート**: 在庫閾値を下回った際のメール通知、および期日超過時の延滞ステータス自動更新（日次ジョブ）。
- **管理者機能**: ダッシュボード（カテゴリ別サマリー・延滞一覧）、ユーザー管理（admin/member ロール）。
- **検索・絞り込み**: キーワード・カテゴリ・ステータス・ユーザーによるフィルタ＋ページネーション。

## Target Use Cases

- 一般ユーザー: 備品一覧閲覧・貸出申請・返却・マイページで履歴確認
- 管理者: 備品・ユーザー管理、貸出承認、ダッシュボードで運用状況把握

## Value Proposition

Devise + Pundit によるロールベース認証・認可で、一般ユーザーと管理者の操作範囲を明確に分離。
Service Object パターンにより、ビジネスロジックをコントローラから切り離し、テスタビリティを確保。
